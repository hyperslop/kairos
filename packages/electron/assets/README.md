Place your app icons here:

- icon.png       — 512×512 or 1024×1024 (app window icon, used by electron-builder)
- tray-icon.png  — 16×16 or 32×32 (system tray icon)

You can generate these from any image using ImageMagick:

    convert my-logo.png -resize 512x512 icon.png
    convert my-logo.png -resize 32x32 tray-icon.png

Or use an online tool like https://www.icoconverter.com/
